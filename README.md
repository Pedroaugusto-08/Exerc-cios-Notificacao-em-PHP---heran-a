# Exercicios-Notificacao-em-PHP---heranca
Outro exercício de herança feito em php!

Neste exercício em anexo, eu usei o padrão das heranças, usei extends
e os require_once para buscar as funções da classe principal, nos testes
eu usei uma função nova __DIR__ que direciona precisamente e de forma mais rápida
o local onde se encontra o arquivo.
